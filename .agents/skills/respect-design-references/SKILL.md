## Princípio

Toda proposta de design cita ≥1 referência viva desta lista. Isso garante inovação baseada em padrões comprovados, não em suposições.

---

## Referências Curadas

### Apple (apple.com)
**O que copiar:** scroll-driven storytelling, bento grids, spacing generoso, typography hierarchy, image sequences, transições cinematográficas  
**Quando usar:** LPs de produto premium, hero com scroll storytelling, seções de features com grande visual  
**Cuidado:** nosso público é B2B — evitar excesso de lifestyle, manter foco em resultado

### Stripe (stripe.com)
**O que copiar:** gradient animations, grid de features, dark mode premium, blending de cores, cards com hover  
**Quando usar:** LPs SaaS B2B, seções de pricing, hero com gradiente azul animado  
**Referência perfeita** para o branding Respect (Navy+Blue+Cyan)

### Linear (linear.app)
**O que copiar:** dark mode excellence, micro-animations suaves, glassmorphism refinado, blur backgrounds  
**Quando usar:** dark mode LP, componentes com backdrop-blur, paleta de blues/grays

### Vercel (vercel.com)
**O que copiar:** minimalismo tipográfico, geometric shapes, grid de logos de clientes, performance-first  
**Quando usar:** seções de prova social (logos), hero minimalista, features em grid

### Framer (framer.com)
**O que copiar:** showcase de Framer Motion, interatividade constante, page transitions, cursor effects  
**Quando usar:** LPs onde animação é o diferencial, demos interativos

### Anthropic (anthropic.com)
**O que copiar:** aesthetic AI-era, tipografia serif+sans mix, tons quentes sobre fundo claro, seriedade com leveza  
**Quando usar:** posicionamento de "IA profissional confiável", LPs de tech consultoria

### Builderall (builderall.com)
**Referência oficial do branding Respect.** Navy+Blue+Cyan, SaaS Enterprise, headlines ALL-CAPS  
**Consultar sempre** para validar paleta e tom visual

### Raycast / Arc Browser / Luma
**O que copiar:** produto tech premium 2025+, glassmorphism, command palette, gradientes sutis, animate on hover  
**Quando usar:** público dev/tech, LPs de ferramenta/SaaS

## Como usar

Ao propor layout/interação, escrever explicitamente:
> "Hero com image sequence no scroll — ref: apple.com/airpods-pro — replicando troca de imagem conforme scrollY vai de 0 a 0.5"

Formato: `[técnica] — ref: [URL] — [o que especificamente está sendo replicado]`
